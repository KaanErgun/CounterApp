# KTS Besmelematik - Counter App

Basit bir React Native sayaç uygulaması.

## Başlangıç

### Gereksinimler
- Node.js 20+
- Android SDK
- Android cihaz (USB veya wireless bağlantı)

### Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Metro bundler'ı başlat
npm start

# Uygulamayı çalıştır
npm run android
```

### Hızlı Başlangıç Script'leri

```bash
# Dev mode'da başlat
./run-dev.sh

# Wireless debug kurulumu
./wireless-debug.sh
```

## Uygulama Özellikleri

- ✅ Sayaç arttırma
- ✅ Sayaç azaltma  
- ✅ Sayaç sıfırlama
- ✅ Dark mode desteği
- ✅ Hermes JS engine

## Debug

VS Code'da **F5** tuşuna basarak debug başlatabilirsiniz.

## Teknolojiler

- React Native 0.82.1
- TypeScript
- Hermes Engine
